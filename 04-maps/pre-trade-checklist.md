# Pre-Trade Checklist

## Definition
The Pre-Trade Checklist is the **daily routine completed before any trade can be taken**. It ensures that every session begins with a structured market read rather than reactive, emotion-driven trading. Following this checklist consistently is one of the highest-leverage habits in this system.

## Rules

### Phase 1 — Before the Asian Session (Sunday/Evening)
- [ ] Check the economic calendar for the week — mark all high-impact news events (NFP, FOMC, CPI)
- [ ] Note the days with news inside [[Kill Zones]] — those are the highest-probability trade days
- [ ] Review the weekly chart: which direction is the weekly [[Higher Timeframe Bias]]?

### Phase 2 — Before London Open (07:30–07:55 GMT)
- [ ] Read the 4H chart — confirm the [[Higher Timeframe Bias]] is still intact or note if a [[Change of Character]] has occurred
- [ ] Mark the [[Asian Session Range]] high and low precisely
- [ ] Mark the previous day's high and low ([[Session Highs/Lows]])
- [ ] Identify the nearest [[Buy-Side Liquidity]] and [[Sell-Side Liquidity]] levels
- [ ] Identify the [[Draw on Liquidity]] for today — where is price most likely going?
- [ ] Mark any unmitigated [[Order Block (Bullish)|OBs]], [[Fair Value Gap|FVGs]], or [[IFVG|IFVGs]] in the area of interest
- [ ] Check [[SMT Divergence]] baseline — note where the correlated pair is relative to the same levels

### Phase 3 — At London Open (08:00 GMT)
- [ ] Watch for a [[Liquidity Sweep]] of the Asian range or prior session levels
- [ ] If a sweep occurs, run the [[Confluences Checklist]] immediately
- [ ] If no sweep by 09:30 GMT, reduce attention — the London setup window is closing
- [ ] If no qualifying setup appears: **do not trade** — wait for NY open

### Phase 4 — At NY Open (13:25 GMT)
- [ ] Re-check the 4H bias — has anything changed since London?
- [ ] Mark the London session high and low (new [[Session Highs/Lows]])
- [ ] Run the same process: watch for sweep, displacement, entry zone
- [ ] Run [[Confluences Checklist]] before any entry

### Phase 5 — After the Session
- [ ] Log all trades taken in the [[Trade Journal Template]]
- [ ] Note missed setups and why they were skipped
- [ ] Rate your execution (discipline, patience, rule adherence)

## Connexions
